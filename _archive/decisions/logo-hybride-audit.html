Verdict

→ Hybride A (lockup horizontal)

Tu n'as pas à choisir entre Piste 01 et Piste 02 — l'hybride prend le meilleur des deux et corrige leurs faiblesses respectives. C'est la combinaison la plus solide pour une marque qui démarre et qui doit tenir 5 ans sans repasser par la case rebranding.

  • Le symbole raconte l'histoire — le double trait du C dit "calque" sans qu'on l'explique. Quand quelqu'un voit ton logo, il pige avant que tu parles.
  • Le wordmark protège la marque — sans le nom écrit, un C seul peut être confondu avec n'importe quelle marque en C (Carrefour, Citroën). Le wordmark verrouille l'identité.
  • Versatilité maximale — favicon 16px = symbole seul, header web = lockup complet, avatar social = symbole inversé navy, signature email = compact. Une marque qui marche partout sans nouveau dessin.
  • Cohérence avec ton brand-book V5 — Stripe, Notion, Linear utilisent exactement ce pattern (symbole + wordmark). Tu restes dans le langage "studio premium" sans copier.
  • Évolutif — dans 2 ans, si tu fais un livre, une appli, un podcast, tu auras déjà un symbole qui peut vivre sans le wordmark.
Risque à anticiper : le C en double trait doit être exécuté finement. Si les deux traits sont trop épais ou trop espacés, ça devient une nouille graphique. Le ratio 2.4 / 1.6 px (épaisseur) + 15 / 18 (rayon) du dessin proposé est calibré. À conserver lors des déclinaisons.

Prochaines étapes si tu valides l'hybride

1. Décliner en 3 versions : full color (navy + violet), monochrome navy (print, signature email), inverted blanc (sur fond foncé).

2. Générer le SVG propre + le favicon (16, 32, 180, 512 px).

3. Remplacer le logo actuel sur le site (header + meta og-image + signature email).
